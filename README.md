This calculator applicaion need nodejs to be install

Installation stpes for ubuntu os:
    sudo apt update
    sudo apt install curl -y
    curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
    sudo apt install nodejs -y
    node -v
    npm -v

    git clone https://github.com/gif-git/calculator.git
    cd calculator
    node server.js
